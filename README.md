# Database Automation Scripts

This repository contains sample scripts and documentation demonstrating database backup, monitoring, and basic maintenance tasks.

## Included Files
- `scripts/backup_monitor.sh`: Bash script for automated database backup and data quality check.
- `Sample_DB_Script_Samuel_Mulandu.docx`: Technical document describing architecture, purpose, and usage.

## Purpose
These files demonstrate:
- Ability to automate backups
- Basic monitoring of data integrity
- Structured logging and documentation
- Understanding of database operations for maintenance
